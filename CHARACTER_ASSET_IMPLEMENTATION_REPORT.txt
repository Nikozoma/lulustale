TECHNICAL REPAIRS
-----------------
The supplied primary PNG exports were 887x1774 RGB files with a visibly baked-in
checkerboard instead of real transparency. Runtime-ready sheets were repaired by:

- removing the connected baked checkerboard and neutral export halo;
- exporting genuine RGBA transparency;
- normalizing the canvas to 888x1776;
- dividing it into a consistent 4-column x 8-direction grid;
- using 222x222 frame cells and four frames per direction;
- preserving character pixels without image resampling or redesign;
- re-centering frames on a stable bottom-center foot anchor.

Direction order is:
  down, down-left, left, up-left, up, up-right, right, down-right

Clear source problems not inventively repaired:
- Lulu pet-dog and knocked-down contain only seven direction rows; down-right is absent.
- Each Charles Jr. worker sheet compresses 16 idle/walk direction rows into an
  eight-row-size canvas. Adjacent poses overlap fixed cells and cannot be safely split
  without cutting or recreating artwork.
- The shadow image is a labeled presentation/reference board, not a regular animation
  sheet.

These seven files received safe transparency/canvas repair only and need corrected
source exports before gameplay use.
